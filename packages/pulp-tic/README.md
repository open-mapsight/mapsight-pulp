# Pulp TIC

Extension for the Pulp library to parse Traffic Information Center (TIC) XML data.

## Features

- **TIC2 & TIC3 Support:** Parse both version 2 and version 3 of the TIC XML format.
- **GeoJSON Conversion:** Transform traffic data into GeoJSON features for easy mapping and analysis.
- **Traffic Event Parsing:** Extract detailed traffic event information from TIC streams.
